# Forensic Audio Authentication

This repository contains Jupyter notebooks for audio forensic authentication using deep learning and machine learning techniques.

## Project Structure

The project is organized into the following stages:

1. **01-data-preprocessing.ipynb** - Data loading, cleaning, and preprocessing steps
2. **02-lfcc-extraction.ipynb** - Linear Frequency Cepstral Coefficients (LFCC) feature extraction
3. **03-model-training.ipynb** - Model training and evaluation
4. **04-xai-gradcam.ipynb** - Explainable AI using Grad-CAM visualization
5. **05-gradio-app.ipynb** - Interactive Gradio application for real-time predictions

## Features

- Audio preprocessing and feature extraction
- LFCC feature computation for audio signal analysis
- Deep learning model training and evaluation
- Explainable AI with gradient-based visualization
- Interactive web interface using Gradio

## Requirements

- Python 3.7+
- Jupyter Notebook
- NumPy, Pandas
- scikit-learn
- TensorFlow/Keras
- Librosa
- Gradio

## Installation

1. Clone the repository
2. Navigate to the notebooks directory
3. Install required packages: `pip install -r requirements.txt`
4. Open Jupyter and run the notebooks in order

## Usage

Run the notebooks sequentially from 01 to 05 to complete the full pipeline.
